BUZZER
- Daily Scheduler

Buzzer has an ability to keep a track of the daily checklist and reminders and the time spent based on the various categories.
Features implemented:
1. Authentication
2. Contains Tabs- Checks, Reminders, Tasks.
3. Ability to add reminders along with the time.
4. Ability to mark/unmark checks and reminders.
5. When the clock hits the reminder time, buzzer alarms.
6. Ability to sort the data as per priority- Critical, high, medium, low.
7. Ability to analyze the data and report in Pie Chart.
